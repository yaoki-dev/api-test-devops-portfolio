### 2. @import/@includeメカニズムの実態

#### 技術的動作（実測・推測）

```python
# @importの動作モデル
class ImportMechanism:
    def process_import(self, main_file):
        # ステップ1: ファイル読み込み
        content = read_file(main_file)

        # ステップ2: @importディレクティブ検出
        imports = find_all_imports(content)

        # ステップ3: インポートファイル展開（全て）
        for import_path in imports:
            imported_content = read_file(import_path)
            content = content.replace(f"@{import_path}", imported_content)

        # ステップ4: 展開済みコンテンツをコンテキストに投入
        return content  # 全コンテンツがコンテキストに入る
```

#### パフォーマンステスト結果

