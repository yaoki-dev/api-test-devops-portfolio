### 1.4 エージェント試算「10-15件限界」の真因

#### 乖離分析

**エージェント試算の前提**:
- 純作業時間: 24.6H（50件）
- 30H残り時間 → 10-15件限界

**逆算による乖離検証**:
- 10-15件 → 1件あたり2-3H
- これは **1件あたり120-180分** を意味する

**真因の特定**:

エージェント試算は以下を **過大評価** していた可能性:
1. **面談Q&A準備時間の配分ミス**
   - 試算: 4H固定
   - 現実: 応募件数に応じて増加（50件 → 8H必要）
   - 過小評価: 4H

2. **品質基準の厳格化による時間増**
   - 高品質文面: 17分/件（試算）→ 実際は25分/件（推敲・調整含む）
   - 過小評価: 8分/件 × 28件 = 3.7H

3. **並列処理の非効率**
   - 試算: リサーチ・文面作成・応募を並列化可能と想定
   - 現実: シリアル処理が主（リサーチ完了→文面作成→応募）
   - 過小評価: 5-8H

**結論**: エージェント試算は **隠れ時間27.4H** を完全に無視し、かつ **純作業時間も過小評価** していた。正確な試算は以下:

| 作業 | 試算値 | 実際値 | 差分 |
|------|-------|-------|------|
| 純作業時間（50件） | 24.6H | 33.5H | +8.9H |
| 隠れ時間 | 0H | 27.4H | +27.4H |
| **合計** | **24.6H** | **60.9H** | **+36.3H** |

**30H残り時間での現実的応募件数**: 60.9H必要 ÷ 50件 = 1.22H/件 → 30H ÷ 1.22H = **24.6件**

### 7.3 用語集

| 用語 | 定義 |
|------|------|
| **表面値** | 純粋な作業時間のみ（隠れ時間を含まない） |
| **隠れ時間** | 作業切替、エラー修正、疲労損失等の見えにくい時間 |
| **真実値** | 表面値 + 隠れ時間（実際にかかる総時間） |
| **疲労補正** | 疲労による生産性低下を考慮した係数（例: 86% = 14%低下） |
| **ROI** | Return on Investment（投資対効果）= 時間短縮 ÷ 投資時間 |
