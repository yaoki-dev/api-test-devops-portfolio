# Claude Flow Lite 実装プロジェクト

## プロジェクト概要

Claude Flow MCPツールを90個から34個に削減し、トークン消費量を62.2%削減（51,300 → 19,380トークン）する実装プロジェクト。

**目的**: Week 7-10で使用する34ツールのみをロードする軽量版Claude Flowパッケージを作成し、セッション起動時のトークン消費を31,920トークン削減する。

**実装期間**: 4時間（または2時間×2セッション）

**目標メトリクス**:
- ツール削減率: 62.2% (90→34)
- トークン削減: 31,920トークン/セッション
- 実装時間: 4時間以内
- エラー率: 0%
- ROI回収期間: 1週間（20セッション）

```bash
# 典型的なエラーチェック

# 2. "Property 'name' does not exist" → allTools型定義確認
# 3. "ALLOWED_TOOL_NAMES is not defined" → import文位置確認
```

**実装時間**: 1時間

**重要ポイント**:
- TypeScriptでは、importパスに.js拡張子必須（ビルド後のJSファイルを指す）
- allTools配列は変更しない（既存90ツール維持）
- フィルタリングロジックのみ追加

**修正例**:
```typescript
server.setRequestHandler(ListToolsRequestSchema, async () => {
  const allTools = [
    // 既存の90ツール定義（変更不要）
  ];

  // ✨ フィルタリングロジック追加
  const filteredTools = allTools.filter(tool =>
    ALLOWED_TOOL_NAMES.includes(tool.name)
  );

  // デバッグログ
  console.error('[claude-flow-lite] Filtered tools:', filteredTools.length);
  console.error('[claude-flow-lite] Expected:', ALLOWED_TOOL_NAMES.length);

  return { tools: filteredTools };  // ← 変更
});
```
