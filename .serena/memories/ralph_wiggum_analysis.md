|------|------|
| 本質 | Stop hookによる `while true` ループでClaudeを自律実行 |
| 成功条件 | 明確な終了条件 + 自動検証可能なタスク |

## プロンプトテンプレート（将来用）

### テスト追加
```bash
/ralph-wiggum:ralph-loop "
【目標】tests/unit/に新規テストファイル追加でカバレッジ{target}%達成

【成功基準】
✅ pytest全通過 ✅ カバレッジ{target}%+ ✅ ruff 0エラー ✅ mypy 0エラー

【制約】
⚠️ conftest.py変更禁止 ⚠️ 新規ファイル作成のみ

【TDD手順】
1. pytest --cov実行 2. 未カバー行特定 3. 新規テスト作成
4. 達成: <promise>COV_TARGET_DONE</promise>

【ブロック時】15イテレーション後に未達なら報告して終了
" --max-iterations 20 --completion-promise "COV_TARGET_DONE"
```

### バグ修正
```bash
/ralph-wiggum:ralph-loop "
【目標】test_xxx の失敗修正
【前提】対象領域カバレッジ70%以上
【成功基準】全テスト通過
【手順】1. pytest -vv 2. 原因特定 3. 修正 4. <promise>BUG_FIXED</promise>
" --max-iterations 15 --completion-promise "BUG_FIXED"
```
