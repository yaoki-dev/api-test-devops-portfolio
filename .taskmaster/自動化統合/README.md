## 🔑 重要概念クイックリファレンス

### autoExecute JSON Schema
- **ファイル**: `.taskmaster/schemas/autoExecute.schema.json`
- **目的**: tasks.jsonのautoExecuteフィールド検証
- **必須フィールド**: enabled, taskKey, schedule
- **オプション**: timeWindow, retryOnFailure, executionHistory

### allowed_tasks.yml セキュリティモデル
- **ファイル**: `.taskmaster/allowed_tasks.yml`
- **目的**: コマンド実行のホワイトリスト
- **構造**: taskKey → command + timeout + allowedTaskIds
- **セキュリティ**: shell=False、タイムアウト強制、Task ID制限
